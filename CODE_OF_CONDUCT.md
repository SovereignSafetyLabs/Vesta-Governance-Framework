# Code of Conduct

## Scope

This repository is an independent research effort maintained by Sovereign Safety Labs.  
Its purpose is analytical clarity and structured technical evaluation.

This Code of Conduct applies to:

- GitHub issues  
- Pull requests  
- Discussions (if enabled)  
- Direct repository-related communication  

---

## Expected Conduct

All participants are expected to:

- Engage in respectful, technically grounded discourse  
- Critique ideas rather than individuals  
- Provide clear reasoning when raising concerns  
- Reference specific artifacts, sections, or TRL classifications when applicable  
- Avoid speculation about intent, motives, or character  

This repository prioritizes analytical rigor over rhetorical persuasion.

---

## Unacceptable Conduct

The following behaviors are not permitted:

- Harassment, personal attacks, or intimidation  
- Derogatory or discriminatory language  
- Bad-faith engagement or repetitive antagonistic commentary  
- Attempts to derail discussions from technical substance  
- Misrepresentation of repository scope, claims, or posture  

---

## Research Posture

This repository is not a community co-development project.  
It is a stewarded research corpus.

Substantive architectural direction remains under maintainer authority.  
Disagreement is welcome. Disruption is not.

---

## Moderation & Enforcement

The maintainer reserves the right to:

- Edit or remove comments that violate this Code  
- Close issues that fall outside repository scope  
- Decline pull requests without obligation to merge  
- Restrict participation if conduct becomes disruptive  

Moderation decisions are final.

---

## Reporting

If you believe conduct in this repository violates this Code of Conduct,  
please contact the maintainer directly via the repository contact information.

---

## Final Note

The objective of this repository is disciplined inquiry into AI governance architectures.  
Professionalism ensures the work remains technically credible and evaluable.
